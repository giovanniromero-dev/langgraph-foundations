# LangGraph Foundations

Hands-on learning repository for core LangGraph concepts in Python.

## Overview

This project collects notebooks and supporting files that walk through the building blocks of LangGraph-based applications, from simple state graphs to tool-using chatbots and debugging workflows.

## Topics Covered

- simple graphs and state transitions
- chatbot construction
- dataclass and Pydantic state schemas
- chain composition with LangGraph
- multi-tool chatbot patterns
- ReAct-style agents
- streaming responses
- debugging and local configuration

## Repository Structure

```text
.

|-- 2-Debugging/

```

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Usage

Open the notebooks in Jupyter or VS Code and run them in order.
